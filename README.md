## Continut Idee

De unde luam intrebarile:
    - Ideea Obiect( intrebare, variante de raspuns )
    - Stocam ideile intr-un vector
    - La inceputul jocului, citim un fisier JSON cu intrebarile si le incarcam in vector
    - La fiecare intrebare, alegem random o intrebare din vector si o afisam
    - Dupa ce raspunde, o scoatem din vector ( ca sa nu mai apara )
    - Intrebarile sunt legate de limbajul Move si blockchain in general



I want to create a block chain clicker game that has these functionalities:


Kahoot Style:
    10 minute timp alocat in total pe zi
    1 intrebarea - 1 minute
    2 intrebarea - 2 minute    
    ...
    Viteza de raspuns a intrebarii ~ cu timpul castigat

Achivements
    bonus 10 intrebari consecutive raspunse corect
    ...
    https://pinata.cloud/

UI:
    Clicking effects
    Label that shows how much time is left:
    Start Clicking -> Click Click
    Timeout -> Clicking is over
    Label -> Cantitatea de coin-uri
    Butoane:
        Connect Wallet
        Collect (Send to Wallet)


Free:
    Ads dupa fiecare quiz
    Reluare a unei singure intrebari
    Design la banut este Silver
    10 minute timp alocat

Pro:
    Ofera sansa de a relua X intrebari
    No Ads
    Desing la banut devine Gold
    15 minute timp alocat
    1.5x monede

## Starting your dApp

To install dependencies you can run

```bash
cd clicker_sui_v0
pnpm install
```

To start your dApp in development mode run

```bash
pnpm dev  
```

## Building

To build your app for deployment you can run

```bash
pnpm build
```

# To start python server
```bash
  cd ./clicker_sui_v0/off_chain/server/server
  source ./venv/bin/activate
  pip install fastapi pynacl uvicorn dotenv
  uvicorn server:app --reload --port 5000
```
